HR Key Setup

Brighter Living Onboarding — one-time administrator utility. Generate the encryption keys that protect candidate packs. Run this once, then keep the private key safe.

1Generate your key pair

This creates an RSA-OAEP 4096-bit key pair in your browser. Nothing is sent over the network.

2Copy the public key

Paste this into the server configuration (HR_PUBLIC_KEY). It is safe to share — it can only encrypt, never decrypt.

3Download your private key

This is the only copy. You will need it to decrypt candidate packs in the HR Dashboard.

Store this file securely. Do not upload it anywhere. You will need it to decrypt candidate packs in the HR Dashboard. If you lose it, packs encrypted with this key pair cannot be recovered.

This page makes no network requests. You can run it offline.
